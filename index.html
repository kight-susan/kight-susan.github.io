Recent Projects

Spotify ‘Like’ Predictions

Using Python and Jupyter Notebook, leveraged Spotify data via an API to transform and clean the data to prepare it for training and testing a k-NN classification supervised learning model. Used in conjunction with a Flask App to make predictions using test parameters. Jupyter Notebook is available here.

Unemployment Rate Time Series Forecasting

Leveraged uni-variate time series data that was explored and fit against various models - linear regression, exponential smoothing, and ARIMA function - to find the best model in R. Model types critiqued by fit, model summary measures, accuracy, residuals (including autocorrelation), and then forecasted.

Foundational Text Analysis of Movie Reviews

Using NLP in Python, performed text processing and mining techniques including polarity and subjectivity analyzed a large dataset to explore dataset and drew key conclusions. Leveraged nltk, pandas, textblob and collections, the Jupyter Notebook file is available here.

Airport Flight On Time Data Analysis

As part of a group project, data analysis and visualization were performed on a dataset from Bureau of Transportation Statistics using R. The data was cleaned and transformed, and then an in-depth analysis performed leveraging several libraries for the analysis and data visualization. The findings are here.

Time Series Sales Forecasting by Strategy Type

Explored sales and advertising data in R and fitted it to a linear regression model evaluating for various diagnostics. Added additional lagged time series data (multi-variate) to improve the model fit and added the various strategy types of data. From the final model made a sales prediction following a particular strategy type, summary provided here.

Patient Readmission Predictions

Created a best performing logistic regression model that described and predicted the probability of a patient being readmitted within 30 days of leaving the hospital. Developed using R and a summary of the procedure for the final model and a non-technical presentation of findings was documented.

Best Model for Customer Churn Predictions

Using data from a telecommunication company, used R with the Rattle library to clean and transform the data before running SVM RBF, Neural Network, Logistic Regression, Decision Tree, Random Forest and Adaptive Boosting models to find the best performing model using AUC amongst others as a determinant. Summary can be found here.

Regression Model Critiqued for Bus Ridership Predictions

Using bus ridership data, I created a regression model using multiple time series variables in R and critiqued the model for fit and accuracy. The R Markdown file can be found here.
