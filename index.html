Recent Projects

Spotify ‘Like’ Predictions

Using Python and Jupyter Notebook, leveraged Spotify data via an API to transform and clean the data to prepare it for training and testing a k-NN classification supervised learning model. Used in conjunction with a Flask App to make predictions using test parameters. Jupyter Notebook is available to view here.

Unemployment Rate Time Series Forecasting

Leveraged uni-variate time series data that was explored and fit against various models - linear regression, exponential smoothing, and ARIMA function - to find the best model using R. Model types were critiqued by fit, summary measures, accuracy, residuals (including autocorrelation), and then a forecast was predicted.

Foundational Text Analysis of Movie Reviews

Using NLP in Python, performed text processing and mining techniques including polarity and subjectivity to analyze a large dataset and drew key conclusions. Leveraged nltk, pandas, textblob and collections for the analysis. The Jupyter Notebook file is available here.

Airport Flight On Time Data Analysis

As part of a group project, data analysis and visualization were performed on a dataset from Bureau of Transportation Statistics using R. The data was cleaned and transformed, and then an in-depth analysis performed leveraging several libraries for the analysis and data visualization. Non-technical findings are here.

Time Series Sales Forecasting by Strategy Type

Explored sales and advertising data in R and fitted it to a linear regression model evaluating for various diagnostics. Added additional lagged time series data (multi-variate) to improve the model fit and added the various data on strategy types. From the final model, a sales prediction was made following a particular strategy - a summary is provided here.

Patient Readmission Predictions

Created a best performing logistic regression model that described and predicted the probability of a patient being readmitted within 30 days of leaving the hospital. This was developed using R and a summary of the procedure for the final model was documented. A non-technical presentation of the findings are available.

Best Model for Customer Churn Predictions

Using data from a telecommunication company, used R with the Rattle library to clean and transform the data before running SVM RBF, Neural Network, Logistic Regression, Decision Tree, Random Forest and Adaptive Boosting models to find the best performing model using AUC amongst other measures as a determinant. A summary can be found here.

Regression Model Critiqued for Bus Ridership Predictions

Using bus ridership data for Iowa City Transit, created a regression model using multiple time series variables in R and critiqued the model for best fit and model accuracy. A R Markdown file of the proceedure that was followed is available.
